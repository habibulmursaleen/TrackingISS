## General info

This project is to create a small app that is tracking the position of the International Space Station in Google maps or similar.

## Technologies

Project is created with:

- ReactJS
- NodeJS
- Postgresql

## Setup

To run this project, install it locally using npm:

### FrontendSetup

```
$ cd ../Frontend
$ npm install
$ npm start
```

### BackendSetup

```
$ cd ../Backend
$ npm install
$ node index.js
```
